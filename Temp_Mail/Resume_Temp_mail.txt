Résumé du script GmailAliasGenerator (en 4 points)

1. Initialisation de l’adresse Gmail
   - L’utilisateur définit une adresse de base (ex. johndoe@gmail.com).
   - Le script vérifie que l’adresse est bien au format Gmail.

2. Génération d’alias Gmail
   - Méthode des points : insertion aléatoire de points dans le nom d’utilisateur.
   - Méthode du suffixe + : ajout d’un suffixe aléatoire ou personnalisé après un "+".

3. Interface interactive en ligne de commande
   - Menu proposant plusieurs options : définir l’adresse, générer des alias, ouvrir Gmail, etc.
   - Les alias générés peuvent être automatiquement copiés dans le presse-papiers.

4. Ouverture de Gmail & explications intégrées
   - Accès direct à Gmail via le navigateur.
   - Explication intégrée sur le fonctionnement des alias Gmail et leurs cas d’usage.
